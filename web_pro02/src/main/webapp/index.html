    시맨틱 태그 Header 머리글에 해당하는 영역을 의미하는 태그(예 : 최상단 메뉴, 검색기능, 로고이미지, ...)
    시맨틱 태그로 본문 내용(컨텐츠)이 들어가는 영역으로 내용들을 주제별로 묶기 위해 사용
    시맨틱 태그로 실제 본문 내용이 들어가는 영역으로 재배포 가능한 내용임을 의미한다.
    시맨틱 태그로 사이드 광고 및 메뉴가 포함된 영역임을 알리는 태그
    시맨틱 태그로 회사 소개, 저작권, 연락처 정보 등의 정보가 있는 영역임을 알린다. 보통 페이지 하단에 배치하여 사용한다.
    시맨틱 태그로 메뉴 네비게이션이 있는 영역임을 알린다.
    위에 작성한 시맨틱 영역 태그는 CSS 와 같이 사용하여 Layout 구성(디자인의 영역)을 만들게 된다.
